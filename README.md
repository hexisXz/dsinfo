# dsinfo

distro information program.


# install 

``` sh
sh <(curl -v -L https://raw.githubusercontent.com/hexisXz/dsinfo/main/install)
```


# usage 

```
-nv: distro name & version

-v: distro version

-s: distro support page

-n: distro name

-k: get kernel version

-h: list program options
```
